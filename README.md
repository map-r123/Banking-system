# Simple Banking System

A small console-based banking application in C++ that demonstrates basic account operations: create an account, deposit, withdraw, and check balance.

## Features

- An account with account number, holder name, and initial balance
- Deposit funds
- Withdraw funds
- Check current balance
- Simple menu-driven CLI
- Support multiple accounts
- Authentication (PIN/password)
- Persist accounts to a txt file
- Input validation and error handling


## Future Enhancements

- Persist transaction history to a file 
- Add transaction history and receipts
